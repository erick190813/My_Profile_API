Python

Python é uma linguagem de programação de alto nível, amplamente utilizada para desenvolvimento web, ciência de dados, aprendizado de máquina e automação. Sua sintaxe simples e legível facilita o aprendizado e a manutenção de código.

Java

Java é uma linguagem de programação orientada a objetos conhecida por sua portabilidade, robustez e segurança. É amplamente utilizada no desenvolvimento de aplicativos empresariais, Android e sistemas de grande escala devido à sua compatibilidade multiplataforma.

JavaScript

JavaScript é uma linguagem de programação que permite a criação de interatividade em páginas web. É essencial para o desenvolvimento de front-end e, com frameworks como Node.js, também é usado no back-end, facilitando a criação de aplicativos web dinâmicos.

HTML

HTML (HyperText Markup Language) é a linguagem padrão para criar e estruturar conteúdos na web. Utilizado para definir a estrutura e o layout de páginas web com elementos como cabeçalhos, parágrafos, links e imagens.

CSS

CSS (Cascading Style Sheets) é usado para estilizar e formatar documentos HTML. Ele controla a aparência de páginas web, permitindo a definição de layouts, cores, fontes e outros aspectos visuais para criar uma experiência de usuário atraente e coesa.

Banco de Dados

Banco de Dados são sistemas usados para armazenar e gerenciar dados de forma eficiente. O MySQL é um sistema de gerenciamento de banco de dados relacional que permite a criação, leitura, atualização e exclusão de dados de maneira segura e estruturada.
